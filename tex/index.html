BibTeX Files

  1. amsalpha2.bst
    Links: Preview | Download
    Description
    I modified the traditional 
    amsalpha.bst
    file (here) to allow for e-prints. Admittedly, I knew absolutely zero about BibTeX styling before this so I borrowed most of my additions from David Dumas and his modification (here) of one of Curt McMullen's stylesheets (here). Drs. Dumas and McMullen both say that this whole chain of editing can be traced back to the classic 
    alpha.bst
    file included with BibTeX, for which I'll take their word since I myself am ignorant of the timeline.
  2. amsalpha2annotate.bst
    Links: Preview | Download
    Description
    I modified my own 
    amsalpha2.bst
    to allow for annotations (for use in annotated bibliographies).
